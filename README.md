COM 103 Midterm Exam: Group Project Task Tracker

This Python tool tracks who is doing what in a 4-person group, monitors individual progress, and calculates the team's total completion percentage.

THIS ARE THE FEATURES

Dynamic Input: Accepts custom project titles and group names.
Task Assignment: Allows exactly 4 assignment slots. Users can assign tasks by number or skip slots by typing 0.
Automated Scoring: Automatically assigns points based on task status (done = 2 pts, pending = 1 pt).
Progress Calculation: Computes total points, max possible points, and whole-number percentage progress.
Status Evaluation: Assigns a project status label based on progress thresholds:
≥ 75% → PROJECT READY!
≥ 50% → ON TRACK.
< 50% → NEEDS MORE WORK!
Formatted Output: Displays a clean, aligned "Task Board" and a detailed mathematical breakdown at the end.

 How to Run

Instructions
Download or clone this repository to your local machine.
Open the folder containing the files.
Open your terminal, command prompt, or VS Code.
Run the program by executing the following command:
python midterm_solution.py
